About This Website

This website is all about writing a static website generator for myself.

I tried to do this a number of years ago but gave up. I couldn’t decide how to write it and more importantly, what I wanted it to do, what its functionality would be. So, I gave up and used Jekyll instead.

My Journal has been written with Jekyll, but I keep hitting limitations.

So, I’ve decided to start again, only this time I’ll be basing it on Jekyll.

The rest of this page gives a basic description of what I hope to achieve. I’ll also have a Blog, where I’ll describe some of the programming techniques I’m using. I’ll also be keeping a diary.

It’s going to take a long time, hopefully I’ll be finishing it around the time .Net 11 is released next year. I’ll be using preview releases of .Net 10 and 11. There will not be a .Net Standard version (too big a limitation).

It will be open-source, and the code will be available as I write it.
